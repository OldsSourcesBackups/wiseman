Wiseman News

Dec 6th, 2007

Wiseman at MDC (http://www.mandevcon.com/). The Wiseman project and the JMX Webservice Adaptor was the topic of a presentation at the Management Developers Conference last week in San Jose, CA.  A discussion of how the adaptor can be used to quickly WS-Management enable java applications that expose management information via JMX. The slides are available online at https://wiseman.dev.java.net/preso/mdc/MDC-2007-Wiseman-JMX.ppt

June 22nd, 2007

The 1.0 binary release of the Wiseman project is now available. Full coverage of the published version of WS-Management is provided along with formalized documentation on how to use this release. Wiseman is a model agnostic implementation of WS-Management that can be used to build powerful web service implementations.

June 1st, 2007

A second Release Candidate for 1.0 of the Wiseman project is available here with the install instructions here. This should be the last release candidate before the finalized binary release of 1.0. Thank you for your interest.

May 7th, 2007

A Release Candidate for 1.0 of the Wiseman project is available here with the install instructions here. After a few additional weeks of performance analysis and incremental updates we hope to put out a finalized binary release of 1.0 in a short time.

March 2nd, 2007

The 0.6 binary release of the Wiseman project now features JAX-WS deployment. We are in the process of adding more support to embrace the newer annotation focused development of web services. We are also renewing efforts to update the documentation of the existing libraries/functionality for the next release. A few enhancements and bug fixes over the previous release have also been added.

Dec 12th, 2006

Wiseman at MDC (http://www.mandevcon.com/). The Wiseman project was the topic of a presentation at the Management Developers Conference last week in San Jose, CA.  A general status and update on what we have implemented was given including details on our client library and upcomming JMX support. The slides are available online at https://wiseman.dev.java.net/preso/mdc/MDC_2006_WSMan_and_Wiseman.pdf

Sept 18th, 2006

The Wiseman project has produced its first binary release! Release 0.5 provides good coverage of the DMTF 1.0a Version of the WS Management specification. This release features a tutorial on creating and exposing resources starting from your schema though generating a Java web application. It also contains sample client and server applications.
