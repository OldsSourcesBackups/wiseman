Installation
  1. create a directory that will be your <Wiseman root directory>, and save the Wiseman bundle to that location.
    2. *In the <Wiseman root directory> defined, execute the following two commands in succession from a command prompt.
  2. jar -xf WISEMAN_0.6_FC-20070302.jar
    3. *You must navigate to the <Wiseman root directory> during this install step.
  3. java -jar WISEMAN_0.6_FC-20070302.jar
