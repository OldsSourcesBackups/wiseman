Developer Documentation

With the 1.0 release we have added a lot more formalized documentation within the <Wiseman Install Dir>/docs directory generated upon installation. In this location you will find:

  • Wiseman Server Developer's Guide.pdf
  • Wiseman Client Developer's Guide.pdf
  • Wiseman Tutorial.pdf
  • zipped API document information
A lot of effort has gone into defining instructions showing how the provided libraries should be used. You should begin here to learn more about the Wiseman framework. Additional direction can be found within the unit tests and source implementation files within CVS.
