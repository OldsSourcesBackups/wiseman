Wiseman 1.0: Release Date: June 22, 2007

Legal Notices
This component is released under the following licenses. You must agree to use these licenses as part of the installation to get a fully functional Wiseman 1.0 installation upon completion.
  • Apache License, Version 2.0
  • COMMON DEVELOPMENT AND DISTRIBUTION LICENSE (CDDL) Version 1.0

Installation
  1. Save the Wiseman install bundle to your local drive.
  2. java -jar WISEMAN_1.0_FC-20070622.jar
  3. Navigate to the new 'wiseman' directory and begin using Wiseman.
Release Notes
Due to bugs uncovered in tools that we depend upon(JDK and ANT releases), you are encouraged to use JDK 1.5.0_12 version(or better) and ANT 1.6.5 for best results.

Please subscribe to the users list and send comments/feedbacks/opinions to the list. Also, bugs that are found in the release can be reported through the issue tracker. Detailed release notes for this 1.0 binary are available here.

Wiseman 1.0_RC2: Release Date: June 1, 2007

Legal Notices
This component is released under the following licenses. You must agree to these licenses as part of the installation to get a fully functional Wiseman 1.0_RC2 installation upon completion.
  • Apache License, Version 2.0
  • COMMON DEVELOPMENT AND DISTRIBUTION LICENSE (CDDL) Version 1.0

Installation
  1. Save the Wiseman install bundle to your local drive.
    2. *In the directory where you saved the install bundle, execute the following two commands in succession from a command prompt.
  2. jar -xf WISEMAN_1.0_RC2-20070531.jar
    3. *Due to a limitation within the installer, you must navigate to the same location where you saved the intall bundle to during this install step.
  3. java -jar WISEMAN_1.0_RC2-20070531.jar
  4. Navigate to the new 'wiseman' directory and begin using Wiseman.
**In the event that you encounter difficulties during installation, please pay close attention to the command line messages as they will provide helpful troubleshooting information.
